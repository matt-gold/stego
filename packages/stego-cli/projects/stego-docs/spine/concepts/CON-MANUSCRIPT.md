# `manuscript/` contains canonical source writing files ordered by filename prefix.

- `manuscript/` contains canonical source writing files ordered by filename prefix.
- Templates consume these files in order and can preserve or restructure that order with helpers such as `splitBy(...)`.
- Related commands: CMD-VALIDATE, CMD-BUILD.
- Related configuration: CFG-REQUIRED-METADATA, CFG-TEMPLATES.
