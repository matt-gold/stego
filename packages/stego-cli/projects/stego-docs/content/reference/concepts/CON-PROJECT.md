# A project is one writing unit inside `projects/<project-id>/` with its own config, sources, and template.

- A project is one writing unit inside `projects/<project-id>/` with its own `stego-project.json`, source folders, and `templates/`.
- Related commands: CMD-NEW-PROJECT, CMD-VALIDATE, CMD-BUILD.
- Related workflows: FLOW-NEW-PROJECT, FLOW-DAILY-WRITING.
