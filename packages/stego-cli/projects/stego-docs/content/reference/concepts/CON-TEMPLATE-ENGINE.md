# The template engine turns project collections into compiled output using TSX and @stego-labs/engine.

- The template engine turns project collections into compiled output using TSX and `@stego-labs/engine`.
- Default projects use `templates/book.template.tsx`.
- Advanced template mode supports target-aware templates, multiple templates per project, and a narrowed Stego API through auto-discovered `templates/*.template.tsx`.
- Related commands: CMD-BUILD, CMD-EXPORT, CMD-TEMPLATE-BUILD, CMD-TEMPLATE-EXPORT.
- Related configuration: CFG-TEMPLATES, CFG-TEMPLATE-COLLECTIONS.
- Related concepts: CON-TARGET-AWARE-TEMPLATES.
