# stego new --project <project-id> [--dir <content-relative-dir>] [--i <prefix>|-i <prefix>] [--filename <name>] [--root <path>]

- `stego new --project <project-id> [--dir <content-relative-dir>] [--i <prefix>|-i <prefix>] [--filename <name>] [--root <path>]`
- Create a new leaf file in the requested content directory. By default Stego suggests a numeric prefix, but prefixes are optional.
