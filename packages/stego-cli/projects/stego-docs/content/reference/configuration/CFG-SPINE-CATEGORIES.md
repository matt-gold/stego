# Stego no longer has first-class spine categories; use content/reference plus metadata instead.

- Organize reference leaves under `content/reference/`.
- Use metadata such as `kind`, `topic`, or `label` when templates or the extension need richer grouping.
- This keeps authored content on one model while still supporting glossaries, source libraries, and browsable reference sections.
