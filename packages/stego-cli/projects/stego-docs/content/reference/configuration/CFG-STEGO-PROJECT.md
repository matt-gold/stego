# Project-level configuration in `stego-project.json` defines metadata rules, image defaults, and other project-specific settings.

- Project-level configuration in `stego-project.json` defines metadata rules, image defaults, and other project-specific settings.
- Build structure itself lives in `templates/`, not in `stego-project.json`.
- Related concepts: CON-PROJECT, CON-METADATA, CON-SPINE, CON-TEMPLATE-ENGINE.
- Related commands: CMD-VALIDATE, CMD-BUILD.
