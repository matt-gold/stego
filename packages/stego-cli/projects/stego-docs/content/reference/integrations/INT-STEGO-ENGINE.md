# `@stego-labs/engine` powers TSX templates and the backend-document pipeline behind Stego builds.

- `@stego-labs/engine` powers TSX templates and the backend-document pipeline behind Stego builds.
- It is installed in scaffolded workspaces so template files get editor support immediately.
- Related commands: CMD-BUILD, CMD-EXPORT, CMD-TEMPLATE-BUILD, CMD-TEMPLATE-EXPORT.
- Related integrations: INT-VSCODE.
